Movie API Documentation

The web application provide's users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints

Request URL HTTP Method Request Body Format Response Body Format
Return All Movies /movies GET None JSON object containing entire movie list.
Return Movie Data by Title /movies/[title] GET None JSON object containing data of single movie. Includes: Title, Year, Description, Genre Name, Genre Description, Director Name, Director Bio, Director Year of Birth, Director Year of Death, and Poster IMG URL.
Return Genre Data by Title /movies/genres/[genreName] GET None JSON object containing data of single genre. Includes: Genre Name and Genre Description.
Return Director Data by Title /movies/directors/[directorName] GET None JSON object containing data of single director. Includes: Director Name, Director Bio, Director Year of Birth, and Director Year of Death.
Register New User /users POST JSON object containing new user's submitted information. Structure: {name: string, email: string} JSON object containing new user's account information, a unique id number, and an empty favoriteMovies array. Structure: {name: string, email: string, favoriteMovies: array, id: string}
Update Username /users/[id] PUT JSON object containing the updated username. Structure: {name: string} JSON object containing user's updated account information. Structure: {name: string, email: string, favoriteMovies: array, id: string}
Add Favorite Movie /users/[id]/[movieTitle] PUT None Text message. Structure: {movieTitle} added to {userName}'s favorites.
Remove Favorite Movie /users/[id]/[movieTitle] Delete None Text message. Structure: {movieTitle} removed from {userName}'s favorites.
Delete User /users/[id] DELETE None Text message. Structure: {userEmail} deleted.
