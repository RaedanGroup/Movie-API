Movie API Documentation

The web application provide's users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints

Request URL HTTP Method Request Body Format Response Body Format
Return All Movies /movies GET None JSON object containing entire movie list.
Return All Users /users GET None JSON object containing entire user list.
Return Movie Data by Title /movies/[title] GET None JSON object containing data of single movie. Includes: Title, Year, Description, Genre Name, Genre Description, Director Name, Director Bio, Director Year of Birth, Director Year of Death, and Poster IMG URL.
Return Genre Data by Name /movies/genres/[genreName] GET None JSON object containing data of single genre. Includes: Genre Name and Genre Description.
Return Director Data by Name /movies/directors/[directorName] GET None JSON object containing data of single director. Includes: Director Name, Director Bio, Director Year of Birth, and Director Year of Death.
Register New User /users POST JSON object containing new user's submitted information. Structure: { name: string, password: string, email: string, birthday: date} JSON object containing new user's account information, a unique id number, and an empty favoriteMovies array. Structure: { username: string, password: string, email: string, birthday: date, favoriteMovies: array, _id: string, __v: number}
User Login /login?username={username}&password={password} POST None JSON object containing user's account information and jwtoken. Structure: { username: string, password: string, email: string, birthday: date, favoriteMovies: array, _id: string, __v: number},"token": string
Update Username /users/[username] PUT JSON object containing the updated username. Structure: {username: string} JSON object containing user's updated account information. Structure: { username: string, password: string, email: string, birthday: date, favoriteMovies: array, _id: string, __v: number}
Add Favorite Movie /users/[username]/favorite/[movieTitle] PUT None Text message. Structure: {movieTitle} has been added to {userName}'s favorites.
Remove Favorite Movie /users/[username]/favorite/[movieTitle] Delete None Text message. Structure: {movieTitle} has been removed from {userName}'s favorites.
Delete User /users/[username] DELETE None Text message. Structure: Account for {userEmail} has been deleted.
